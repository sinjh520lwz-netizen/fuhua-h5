+## 第一章：收益率的数学本质——从正态分布到厚尾
+
+### 1.1 收益率的定义与选择
+
+在回测中，我们首先面临一个看似简单但影响深远的选择：用哪种收益率？
+
+**简单收益率（Simple Return）**：
+$$R_t = \frac{P_t - P_{t-1}}{P_{t-1}} = \frac{P_t}{P_{t-1}} - 1$$
+
+**对数收益率（Log Return）**：
+$$r_t = \ln\left(\frac{P_t}{P_{t-1}}\right) = \ln(P_t) - \ln(P_{t-1})$$
+
+两者的关系（用Taylor展开）：
+$$r_t = \ln(1 + R_t) = R_t - \frac{R_t^2}{2} + \frac{R_t^3}{3} - \cdots$$
+
+当 $|R_t|$ 很小（比如日收益率 < 5%）时，$r_t \approx R_t$，差异可以忽略。但对JH系统的TP+5%/SL-4%策略，单笔收益可能达到5%，此时 $r = \ln(1.05) = 0.0488$，与 $R = 0.05$ 的差距约2.4%——不大，但在600+笔交易中累积起来就不可忽视。
+
+**为什么学术文献偏爱对数收益率？**
+
+数学原因：对数收益率具有**时间可加性**。
+$$r_{t_1 \to t_n} = \sum_{i=t_1}^{t_n} r_i$$
+
+简单收益率不具备这个性质：
+$$R_{t_1 \to t_n} = \prod_{i=t_1}^{t_n}(1 + R_i) - 1 \neq \sum R_i$$
+
+这意味着：如果用对数收益率，我们可以在正态分布假设下直接做统计推断（因为正态分布的和仍是正态分布）。而简单收益率的乘积结构使推断变得复杂。
+
+**为什么JH系统应该用简单收益率？**
+
+1. **用户理解成本**：+5%对用户来说直观，0.0488不直观
+2. **回测计算**：总收益 = Σ(单笔收益) 在简单收益率下**不精确**，但JH系统的每笔交易独立（每天选新票，不复合），所以 Σ(单笔简单收益率) 是合理的近似
+3. **TP/SL的自然表达**：TP+5%就是 $R = 0.05$，用对数收益率反而绕弯
+
+### 1.2 正态分布假设的检验
+
+现代金融理论的基石之一是"收益率服从正态分布"（或至少近似服从）。这直接决定了Sharpe比率等指标的统计性质。让我们用实际数据检验这个假设。
+
+**正态分布的特征**：
+- 偏度（Skewness）= 0
+- 峰度（Kurtosis）= 3（超额峰度 = 0）
+
+**A股日收益率的实际特征**（基于大量实证研究）：
+- 偏度：通常为 **-0.3 ~ -0.8**（左偏，大跌比大涨频繁）
