+# 第一部分：因子模型的矩阵表述与数学深化
+
+## 1.1 从评分公式到矩阵方程
+
+前一轮我们用的综合评分公式是：
+```
+Score_i = Σ(w_j × factor_rank_{i,j})
+```
+
+这是一阶近似。把它写成矩阵形式：
+
+设 X ∈ R^{N×K} 为因子矩阵（N只股票，K个因子），w ∈ R^K 为权重向量，则：
+```
+s = Xw
+```
+其中 s ∈ R^N 为各股票的综合得分向量。
+
+### 自我质疑：线性模型够用吗？
+
+问题：真实的因子-收益关系几乎肯定不是线性的。比如小市值因子在极端小盘股上的效应可能是非线性凸函数——越小越猛，但到某个阈值后壳价值溢价反而衰减。
+
+反驳：线性模型在截面预测中的表现往往出人意料地好。原因是：
+1. 因子预处理阶段（标准化、去极值）已经压缩了非线性范围
+2. 截面排序对单调变换不敏感——只要因子与收益的关系是单调的，线性排序分和非线性排序分选出的股票子集几乎相同
+3. 非线性模型的过拟合风险远大于线性模型
+
+自我修正：对于A股这种因子有效性快速变化的市场，线性模型的"够用"有时间窗口。在因子拥挤度低的阶段，线性近似足够；在因子拥挤/风格切换阶段，非线性交互项变得重要。实践中建议：**主线用线性模型，定期用树模型（LightGBM）检测非线性交互是否显著改善样本外IC，如果改善<15%则保持线性。**
+

+## 1.2 因子协方差矩阵与组合风险分解
+
+### 1.2.1 组合收益的方差表达
+
+设组合权重向量 h ∈ R^N（h_i 为第i只股票的持仓金额占比），组合收益：
+```
+R_p = h^T r = Σ h_i r_i
+```
+
+组合方差：
+```
+Var(R_p) = h^T Σ h = Σ_i Σ_j h_i h_j Cov(r_i, r_j)
+```
+
+其中 Σ ∈ R^{N×N} 为收益率协方差矩阵。
+
+### 1.2.2 因子模型对协方差矩阵的降维
+
+直接估计 N×N 协方差矩阵（A股~5000只股票）需要估计约 1250万个参数，严重过拟合。因子模型将其降维：
+
+**Barra结构化因子模型**：
+```
+r = Xf + ε
+```
+其中：
+- f ∈ R^K 为因子收益率向量（K << N）
+- X ∈ R^{N×K} 为因子暴露矩阵
+- ε ∈ R^N 为特异性收益（idiosyncratic return），假设 Cov(ε_i, ε_j) = 0 for i≠j
+
+则协方差矩阵分解为：
+```
+Σ = X F X^T + Δ
+```
+其中：
+- F ∈ R^{K×K} 为因子协方差矩阵（只需估计 K(K+1)/2 个参数）
+- Δ = diag(σ²_ε1, ..., σ²_εN) 为特异性方差对角矩阵
